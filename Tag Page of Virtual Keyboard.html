TAGS USED

1.FAVICON

The link element with the rel attribute set to "icon" is used to define a Favicon for a web page. Favicons are small icons typically displayed in the browser's address bar, next to the page title, and in browser tabs when a user visits a website. They serve as a visual identifier for the website and help users quickly recognize and access it.
SYNTAX: <"link rel="icon" href="favicon.ico" type="image/x-icon">

2.NAVBAR

A navbar, short for "navigation bar," is a user interface element commonly found in websites and applications. It is a crucial component for providing navigation and helping users move around a website or application easily.


HERE ARE SOME KEY FEATURES FOUND IN NAVBAR:

1) Navigation Links: Navbars typically contain links to important sections or pages of a website or application. These links can include items like "Home," "About Us," "Services," "Contact," and so on.
2)Dropdown Menus: For websites with a lot of content or subsections, navbars often include dropdown menus that expand when users hover over or click on a parent menu item.
3)Search Bar: Some websites incorporate a search bar into the navbar, enabling users to search for specific content or products directly from the header.

3.DIVISION

The div tag, short for "division," is one of the fundamental HTML elements used to structure and organize the content of a web page. It is a block-level element, which means it typically starts on a new line and stretches the full width of its container. The primary purpose of the div tag is to group and style content or elements together, making it easier to apply CSS styles and layout to specific sections of a webpage.


HERE ARE SOME KEY FEATURES FOUND IN DIV

1)Content Grouping: div tags are often used to group related content together. For example, you can use a div to wrap a set of paragraphs, headings.
2)Class and ID Attributes: You can add class and ID attributes to div elements to uniquely identify them or apply CSS styles. Classes are used for multiple elements with similar styles, while IDs should be unique within a page.
3)Block-Level Element: div is a block-level element, meaning it typically starts on a new line and extends the full width of its container. This characteristic helps with layout and organization.

4.ANCHOR

The anchor tag is denoted by a within angular brackets, is an essential HTML element used to create hyperlinks or links on web pages. Anchor tags are used to navigate between different web pages, link to specific sections within a page, or trigger various actions when clicked.

KEY ASPECTS OF ANCHOR TAG

1)Href Attribute: The most critical attribute of the anchor tag is href (Hypertext Reference), which specifies the destination of the link. It can point to:
URLs
Relative Paths
Anchors
2)Link Text: The text or content between the opening and closing anchor tags is the visible link text that users click on.
3)Target Attribute: The target attribute allows you to specify how the linked resource should be displayed when the user clicks the link.

5.SECTION

The section tag is an HTML5 structural element used to define sections or thematic groupings of content within a web page. It is part of a set of semantic elements introduced in HTML5 to provide a more meaningful structure to web documents. The section tag is particularly useful for organizing content hierarchically and improving the accessibility and search engine optimization (SEO) of a webpage.

KEY POINTS ABOUT SECTION TAG

1)Semantic Structure: The primary purpose of the section element is to mark up content that represents a distinct thematic grouping or section within a webpage. This helps browsers, search engines, and assistive technologies understand the organization and hierarchy of content.
